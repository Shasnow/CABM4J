## 运行环境
- Java 17+
- Gradle 8.0+

## 使用方法
1. 克隆本项目到本地或下载源代码。
2. 在项目根目录下打开终端。
3. 运行以下命令以构建项目：
   ```bash
   ./gradlew build
   ```
4. 运行以下命令以启动应用程序：
   ```bash
   ./gradlew bootRun
   ```
   或者使用 IDE 运行 `Cabm4JApplication.java` 类。
